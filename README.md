## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* ruby 2.7.0
```sh
rvm install "ruby-2.7.0"
```
* rails 6.0.3.4
```sh
gem install rails
```
* Node 10.22.0
```sh
nvm install 10.22.0
```
* RethinkDB 2.4.0
```sh
brew install rethinkdb
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/tmanhkha/rethinkdb-rails.git
```
1. Change directory
```sh
cd rethinkdb-rails
```
3. Bundle install
```sh
bundle install
```
4. Yarn install
```sh
yarn install
```
5.  Start RethinkDB
```sh
rethinkdb
```
5.  Rails sverver
```sh
rails server
```
